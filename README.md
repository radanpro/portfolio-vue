# RADAN | Digital Architect Portfolio 🚀

A high-end, interactive portfolio built with **Nuxt 4** and **Tailwind CSS 4**, focusing on advanced UI/UX animations, WebGL, and fluid motion.

## 🛠 Tech Stack

- **Framework:** [Nuxt 3/4](https://nuxt.com/) (Future Compatibility Mode)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (CSS-first engine)
- **Animations:** [GSAP](https://gsap.com/) (ScrollTrigger, Custom Easing)
- **Smooth Scroll:** [Lenis](https://github.com/darkroomengineering/lenis)
- **3D Graphics:** [Three.js](https://threejs.org/) (Custom Particle Shaders)
- **Icons:** [Remix Icon](https://remixicon.com/)

## ✨ Key Features

- **Gooey Cursor:** A custom liquid-physics cursor that reacts to movement and interaction.
- **Magnetic Interaction:** Reusable Vue components for magnetic UI elements.
- **3D Perspective Cards:** Project cards with real-time mouse-tracking tilt effects.
- **Particle Background:** A high-performance WebGL background rendered via Three.js.
- **Architectural Layout:** A minimalist, bold typography-focused design inspired by Awwwards.

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS)
- Yarn or NPM

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/radanpro/portfolio-vue.git
   ```
2. Install dependencies:
   ```bash
   yarn install
   ```
3. Start the development server:
   ```bash
   yarn dev
   ```

## 📂 Project Structure

- `app/components/`: Reusable UI components (MagneticButton, ProjectCard, etc.)
- `app/composables/`: Animation logic and global hooks (useMagnetic, useSmoothScroll).
- `app/assets/css/`: Global Tailwind 4 styles and variables.
